# Business Management Platform

Planeacion de un MVP SaaS para control interno de clientes, facturas, pagos y saldos pendientes.

El proyecto esta definido para construirse con Next.js, TypeScript, shadcn/ui, Tailwind CSS, Supabase, Vercel y una API documentada con OpenAPI/Swagger.

## Estado actual

Esta etapa contiene documentacion funcional y tecnica. Todavia no hay codigo de aplicacion implementado.
